# Next Task

Find and display the next task to work on.

1. Read `BUILD_PLAN.md`
2. Find the first unchecked task (`- [ ]`)
3. Show:
   - Phase name and goal
   - Task ID and description
   - Acceptance criteria
   - Dependencies (what must exist before this task)
   - Estimated units of work (how many RED→GREEN→REFACTOR cycles)

4. Ask: "Ready to start? Run `/project:build Task X.Y` or give me different instructions."
